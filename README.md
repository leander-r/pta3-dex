# Pokémon Tabletop Adventures – Character Manager

A fan-made, non-commercial web application for managing trainers, Pokémon, stats, moves, and game data for the **Pokémon Tabletop Adventures (PTA)** tabletop RPG system.

This tool is designed to replace spreadsheets and manual note-taking with a clean, interactive interface that makes long-running PTA campaigns easier to manage.

---

## ✨ Features

- Trainer and Pokémon character management
- Automatic stat calculations based on level, nature, and features
- Pokémon data browsing (types, abilities, moves, skills, evolutions)
- Move lists with typing and metadata
- Class, feature, and skill reference data
- Responsive UI designed for long play sessions
- Fully client-side (no backend required)

---

## 🛠 Tech Stack

- **JavaScript**
- **React** (UMD build, client-side)
- **HTML / CSS**
- **JSON-based data modeling**

AI tools were used as a productivity aid during development (iteration, refactoring, and UI improvements), with all architecture and final implementation decisions made by the author.
